# ReactReduxCountryWebApp
Yobetit Preliminary Front-end Task

Installation

* Clone the repository
* type "cd clientApp" into the terminal and "npm install"
* type "cd server" into the terminal and "npm install"

* go into the clientApp directory again onto the terminal and type "npm start"
* into a separate terminal, go into the server directory and type "nodemon"

The front-end should be located on localhost:3000 and the Node.JS server should be running on port 5000.

The project is divided into each of the 4 questions, with the sidebar on the left in order to provide routing into each question.
